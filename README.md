# finanzas-personales

Proyecto para controlar mis ingresos y gastos personales.

## Estructura del proyecto

- `gastos/` — Script para registrar y guardar gastos en un CSV
- `ingresos/` — Script para registrar ingresos (próximamente)
- `deudas/` — Script para dar seguimiento a deudas propias y de terceros (próximamente)
- `data/` — Carpeta local donde se guardan mis archivos CSV con datos reales (no se sube a GitHub por privacidad)

## Cómo funciona

Cada script genera y guarda registros en un archivo CSV, incluyendo fecha, categoría y monto.

## Tecnologías

- Python
- Módulo `csv` de la librería estándar

## Estado del proyecto

🚧 En desarrollo — actualmente solo el módulo de gastos está implementado.
